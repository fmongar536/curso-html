Objetivos de la unidad

  • Comprender el funcionamiento básico de una página web y su estructura técnica.
  • Aprender el lenguaje HTML y sus etiquetas fundamentales.
  • Desarrollar habilidades para estructurar y presentar información digitalmente.
  • Fomentar la creatividad y la autonomía en el diseño web.
  • Introducir nociones básicas de diseño visual con estilos CSS.
  • Desarrollar una web personal que refleje los aprendizajes adquiridos.

Metodología

La unidad combina explicaciones teóricas breves, aprendizaje práctico guiado, ejercicios progresivos y un proyecto final. Se promueve el trabajo tanto individual como colaborativo.

Requisitos previos

No se requiere experiencia previa en programación. Solo un ordenador con navegador web y editor de texto (recomendado: Visual Studio Code o editor online como jsbin).

Evaluación

  • Ejercicios prácticos: Cada bloque incluirá una serie de prácticas sencillas.
  • Actitud, participación, cooperación, proactividad y resiliencia: Valoración del interés y seguimiento.
  • Proyecto final: Página web personal funcional, estructurada y con estilo básico.

Se utilizará una rúbrica de evaluación basada en el uso correcto del HTML, organización del contenido y creatividad.
